Weather Checker is a vanilla JavaScript web app that fetches real-time 7-day weather forecasts for any city in the world. Built using the Open-Meteo and Geocoding APIs, it converts a city name into coordinates and retrieves daily temperature and weather conditions displayed as emoji icons.
Features

Search any city worldwide
7-day weather forecast
Weather icons based on WMO weather codes
Clean glassmorphism card design

Technologies

HTML, CSS, JavaScript
Open-Meteo Forecast API
Open-Meteo Geocoding API
